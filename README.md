# Arena-Additive-Calculator
Arena Dust Control Blend Calculator
===================================
A smart, data-driven tool for generating custom dust-control fluid formulations for equestrian arenas.
Designed for future integration with automated fulfillment and dropshipping, this calculator tailors
blends using:
• Arena size (square footage)
• Footing type (M10, washed sand, fiber, GGT, etc.)
• Lab/field metrics including particle size, fines %, shear, compressibility, and residual salt level
It outputs total recommended gallons and a component breakdown for:
• Base Oil Blend – primary dust suppression
• Conditioning Ester – improves flow and reduces greasiness
• Polymer Stabilizer (optional) – light binding for very fine or loose footing
Goal: Reduce or eliminate arena watering using non-evaporating, hoof-safe dust control fluids.

How to Use
----------
1) Enter arena length and width (in feet).
2) Enter treatment depth (in inches).
3) Select footing type.
4) (Optional) enter lab test values:
• Average particle size (microns)
• Percent fines below 75µm
• Shear rating (1–10)
• Compressibility (1–10)
• Existing CaCl2 salt level
5) Submit for a custom recommended blend.
------------------------------------------------------------
Scientific Basis (Summary)
--------------------------
Dosing model factors:
• Higher fines or smaller particles increases fluid requirement.
• Deep or compressible footing reduces polymer content to avoid stiffening.
• Residual CaCl2 slightly reduces dose as it provides some moisture retention.
• Large arenas scale total fluid volume automatically.
Typical dosing range:
0.006 to 0.013 gallons per square foot of treated surface.
